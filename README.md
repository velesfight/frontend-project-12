## Analogue of the Slack chat
A simple version of Slack chat in React/Redux, using AJAX, REST React (with hooks), Redux (@reduxjs/toolkit), Formik

## Installation::
Install the dependencies:

```
make install
```
Start:
 ```
 make start
 ```
